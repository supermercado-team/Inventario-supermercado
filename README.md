# Inventario-supermercado

Sistema de gestión de inventario y ventas para supermercado.

## Estructura del proyecto

- `index.html` - Interfaz del sistema de inventario.
- `styles.css` - Estilos para la interfaz.
- `app.js` - Lógica del sistema: proveedores, inventario, alertas y pedidos.
- `.gitignore` - Archivos y carpetas ignorados.

## Funcionalidades implementadas

- Gestión de proveedores: registrar, editar, eliminar y buscar proveedores.
- Control de inventario: registrar productos, entradas, salidas, actualizar stock y mostrar existencias.
- Alertas de stock mínimo: definir stock mínimo global, detectar productos críticos y agotados.
- Gestión de pedidos: crear pedidos con proveedor, producto, cantidad y estado; consultar historial.

## Cómo usar

1. Abrir `index.html` en el navegador.
2. Registrar proveedores, productos y pedidos.
3. Configurar el stock mínimo en la pestaña `Alertas`.
